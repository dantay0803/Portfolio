After reading this post from Wes Bos on what hardware and software he uses and then discovering on Twitter he has a collection of similar posts from other developers. Which he has now turned into a dedicated website using Gatsby. I thought it would be enjoyable to do one of my own, just covering what tools I use when developing personal projects.

<p><a href="https://visualstudio.microsoft.com/vs/preview/">Visual Studio 2019 Community</a> – This is my go-to editor for any of my C# based work. I know you can use VS Code for C# as well, but I feel more at home with Visual Studio and its C# experience is better.</p>

<p><a href="https://github.com/tonsky/FiraCode">Fira Code</a> – In both VS Code and VS Studio I use Fira Code as my font of choice.</p>

<p><a href="https://www.microsoft.com/en-us/p/windows-terminal-preview/9n0dx20hk701?activetab=pivot:overviewtab">Windows Terminal</a> – I recently switched over to using the new Windows Terminal as my terminal of choice.</p>

<p><a href="https://www.mozilla.org/en-GB/firefox/developer/">Firefox Developer Edition</a> – At the time of writing this blog I have recently swicthed from using Chrome to Firefox as my daily browser and I am currently really enjoying the features it brings.</p>

<h2>Desktop Programs</h2>

<p><a href="https://www.adobe.com/uk/products/xd.html">Adobe XD</a> – Allows you to easily and quickly mockup design ideas. You can also find projects of prebuilt components from the likes of Google that you can use.</p>

<p><a href="https://www.getpaint.net/">Paint.Net</a> – For when I need to do some very light image editing.</p>

<p><a href="https://www.microsoft.com/en-gb/microsoft-365/onenote/digital-note-taking-app">OneNote</a> – Another piece of software I have recently started using and have fallen in love with for taking notes and scribbling down ideas.</p>

<p><a href="https://notepad-plus-plus.org/">Notepad++</a> – A Nice powerful editor that you can do a lot with. Whether it be coding or just simple text editing, which is my main use.</p>

<p><a href="https://desktop.github.com/">GitHub Desktop</a> – May be blasphemy to some, but it is very quick and easy to use and since I am just working on tutorials and side projects it does the job for me.</p>

<p><a href="https://www.postman.com/">Postman</a> – For viewing API calls</p>

<p><a href="https://obsproject.com/">OBS</a> – Screen capture.</p>

<p><a href="https://www.blackmagicdesign.com/uk/products/davinciresolve/">DaVinci Resolve</a> – Video editing.</p>

<h2>Web Apps</h2>

<p><a href="https://www.google.co.uk/docs/about/">Google Docs</a> – To me this is a great free alternative to MS Office, I use sheets the most for saving tutorial links and project ideas.</p>

<p><a href="https://jsoneditoronline.org/">JSON Editor Online</a> – Great online JSON formatter and editor.</p>

<p><a href="https://carbon.now.sh/">Carbon</a> – Easy way to get clean screenshots of code snippets</p>

<p><a href="https://1password.com/">1Password</a> – Password manager</p>
